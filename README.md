Smart Expense Tracker

This is a command-line based Python application used to track and analyze daily expenses. The project is designed to help users understand their spending habits.

Users can add expenses by entering date, category, amount, and description. All data is stored in a CSV file, which allows persistent storage without using a database.

The application also provides features to view all expenses, calculate monthly spending, analyze category-wise expenses, and identify the highest spending category.

This project uses Python concepts such as file handling, dictionaries, loops, and functions. The CSV module is used to manage structured data.

Through this project, I learned how to handle data storage in files, process data, and generate useful insights using simple logic.
